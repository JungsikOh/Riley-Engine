# Riley-Engine

## Install
```
$vcpkg install directxtk:x64-windows
$vcpkg install directxmath:x64-windows
$vcpkg install assimp:x64-windows
```

## Engine Features
- Sparse-Set ECS (entt ECS)
- ImGui GUI

## Graphics Features
- Shadow Mapping
- Point Shadow Mapping
- View Frustum Culling
- Deferred Lighting
- SSAO
- AABB
